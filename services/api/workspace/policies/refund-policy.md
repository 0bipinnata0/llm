# 退款政策

## 适用范围
所有通过平台支付成功的订单，均适用本退款政策。

## 退款类型
1. **原路退款**：退回至原支付账户，通常需要 3-7 个工作日到账。
2. **余额退款**：退回至平台账户余额，可立即到账，可用于下次消费。
3. **部分退款**：针对订单中的部分商品或部分金额进行退款，剩余金额继续保留。

## 退款时效
- **未发货订单**：用户申请退款后，系统自动审核，1 个工作日内原路退回。
- **已发货但未签收**：拦截成功后，仓库确认退回，3 个工作日内退款。
- **已签收订单**：需按照退货政策完成退货流程，仓库验收后 3 个工作日内退款。

## 不可退款情形
- 已使用或消耗的虚拟商品、充值卡。
- 超过退款申请时效的订单。
- 因用户个人原因导致商品损坏的订单。

## 退款查询
用户可在「我的订单」中查看退款进度，如有疑问请联系平台客服。
